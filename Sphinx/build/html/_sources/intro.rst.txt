Getting Started
---------------
 Institution Internal Transaction System - utpWallet (*Tentative Name) 
 Developed in Python 3 from Anaconda for Backend, MySQL Community + MySQL WorkBench for Database, PyQt for Graphical User Interface.


Built With / Softwares used
---------------------------
 * `Anaconda - Python3 <https://www.anaconda.com/>`_ - Backend
 * `PyQt from Anaconda <https://riverbankcomputing.com/software/pyqt/intro/>`_ - Python GUI Module
 * `Qt Designer <https://riverbankcomputing.com/software/pyqt/intro/>`_ - Qt tool for GUI design and built
 * `MySQL Community <https://dev.mysql.com/downloads/mysql/>`_ - DBMS & Local Server Setup
 * `MySQL Workbench <https://dev.mysql.com/downloads/workbench/>`_ - SQL Development, Data Modeling, EER Diagram

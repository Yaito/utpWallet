Getting Started
---------------
Our vision with this project is to have a simple payment system in all the facilities where transactions are made with money (cafeteria, library, cashier). This project focuses only on the payment system of registration in the box and the recharge of the account.


What it can do
--------------
 * Tuition payment
 * University wallet top-up
 * Admin login
 * Create Accounts
 * Delete Accounts
 * Show student info
